# Vant-ui
this is vant components extend  
这是vant组件库的扩展

### [Online demo](http://moerj.com/vant-ui/#/demo-markdown)

### 特性
- 更完善的组件库
- 高级选择器组件
- 图片画廊
- 多级菜单
- 支持无限层级页面

### 安装
```node
npm i vant-ui
```
> vant-ui内部已自动安装vant,请勿重复安装vant

main.js中引入
```js
import vantUI from 'vant-ui';
Vue.use(vantUI);
```

### 浏览器支持
现代浏览器以及 Android 4.0+, iOS 6+
