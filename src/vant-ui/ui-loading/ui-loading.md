#局部区域loading

##调用方式

```html
    <ui-loading title="加载中..." background="rgba(255,255,255,0.8)"></ui-loading>
```

##参数

props       |  type    | default  |  说明
------------|----------|----------|----------
title       | String   | ''       |  标题
background  | String   | rgba(255,255,255,0.8) |  背景蒙层颜色
