# 复选框

## 调用方式

```html
<ui-checkbox v-model="checked" name="inputName" label="labelString"></ui-checkbox>
```

## 参数

props       |  type    | default  |  说明
------------|----------|----------|----------
v-model     | Boolean  | false    |  初始是否选中
name        | String   | ''       |  input name
label       | String   | ''       |  标题
color       | String   |          |  选项框的颜色值
